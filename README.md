# FLAC to MP3 Converter

A simple Python command-line tool that converts **FLAC** (lossless) audio files into **MP3** format.  
This small hobby project was created to automate converting music into a format playable in a car stereo that only supports MP3.

## Features

- Converts `.flac` files to `.mp3`
- Batch processing for all files in a directory
- Simple and lightweight command-line interface
- Uses `ffmpeg` through `pydub` for reliable audio conversion
